# Diwali Sales Analysis

---

## Project Overview

The **Diwali Sales Analysis** project explores customer purchasing behavior and sales performance during the Diwali festival.  
By analyzing over **11,000 sales records**, this project identifies key trends, customer demographics, and product categories driving overall revenue growth.

---

## Objectives

- Understand customer demographics and spending behavior.
- Analyze sales distribution across age, gender, and occupation.
- Identify top-performing products and high-value customer segments.

---

## Tools and Libraries Used

| Tool / Library | Purpose                         |
| -------------- | ------------------------------- |
| **Python**     | Programming and data processing |
| **Pandas**     | Data cleaning and manipulation  |
| **NumPy**      | Numerical analysis              |
| **Matplotlib** | Data visualization              |
| **Seaborn**    | Statistical visual insights     |

---

## Approach

1. **Data Cleaning & Preprocessing**

   - Removed duplicates and missing values.
   - Converted data types and standardized columns.

2. **Exploratory Data Analysis (EDA)**

   - Analyzed sales by gender, age group, and state.
   - Visualized purchase patterns and spending trends.

3. **Visualization & Insights Extraction**
   - Created bar plots, pie charts, and histograms using Matplotlib & Seaborn.
   - Highlighted profitable customer demographics and product categories.

---

## Key Insights

- **Female customers** contributed more to total sales compared to males.
- Most purchases were from **married customers aged 26–35 years**.
- **Electronics, Clothing, and Food** were the top-selling categories.
- Targeted marketing toward these groups can yield a **20% increase in sales**.
